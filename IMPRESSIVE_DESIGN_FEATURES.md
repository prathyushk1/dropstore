## ğŸ¨ How to Customize:

### Change Colors:
Edit `tailwind.config.ts` or use different gradient classes:
```tsx
// Current: from-purple-600 to-pink-600
// Change to: from-blue-600 to-cyan-600
```

### Adjust Animations:
Edit `app/globals.css` animation durations:
```css
/* Current: 0.8s */
animation: fade-in-up 0.8s ease-out;
/* Change to: 1.2s for slower */
animation: fade-in-up 1.2s ease-out;
```

### Modify Hover Effects:
Change scale, shadow, or transform values:
```tsx
// Current: hover:scale-105
// Change to: hover:scale-110 for more dramatic
```
